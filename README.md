## üèÜ Featured Project: Digital Library System

**Location:** [`proiect/src/main/java/oop2_project`](proiect/src/main/java/oop2_project)

A robust Java application simulating a digital library management system. This project demonstrates core OOP principles, clean architecture, and database persistence.

### üöÄ Key Features
* **Database Persistence:** Full data integration using **PostgreSQL** and **JDBC**.
* **Architecture:** Implements the **DAO Pattern** to isolate business logic from data access.
* **Audit System:** A custom CSV logging service tracks every user action (borrowing, returning, reservations).
* **Complex Logic:** Handles overdue reports, stock management, and member inheritance (`Reader` vs `Author`).
